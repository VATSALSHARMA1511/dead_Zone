02 / 05

RAG Persistent Memory

Each completed run is converted to natural language, embedded via sentence-transformers, and stored in ChromaDB. Before each wave request, the top 3 most similar past runs are retrieved and injected into the prompt — the AI genuinely remembers how you play.
